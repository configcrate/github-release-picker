# v0.1.0 — First public release

GitHub Release Picker helps people choose the right file from a GitHub release without decoding asset names themselves.

## Included

- Paste `owner/repository`, a repository URL, or a tagged release URL.
- Windows, macOS, Linux, x64, and ARM64 selection.
- Cautious asset ranking with confidence labels.
- Clear handling for checksums, signatures, source-only releases, incompatible builds, missing releases, rate limits, and network errors.
- Chinese and English interface.
- Responsive, dependency-free static web app.
- Direct GitHub downloads with no login, API key, backend, or tracking.

Use the hosted app: https://configcrate.github.io/github-release-picker/

---

GitHub 下载选择器可以替普通用户从 GitHub 发行版的多个文件中找出适合自己设备的那个。第一版支持中英文、Windows/macOS/Linux、x64/ARM64，以及源码、校验文件和不兼容版本提示。无需安装、登录或 API Key。
